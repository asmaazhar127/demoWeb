== Description ==

Building Contractor WordPress is a professional website template designed especially for construction companies, contractors, and builders, providing a robust online presence to showcase projects and services. With its user-friendly design and practical features, this theme caters to businesses in the construction industry, delivering a visually appealing and informative platform. This theme exudes a sense of reliability and expertise with its clean layout, high-quality imagery, and a color scheme that resonates with the construction sector. This theme is specifically designed for Construction, Contractor, Building, Renovation, Real Estate, Development, Architecture, Project Management, Home Improvement, General Contractor, Property Management, Residential, Commercial, Construction Services, Builder. The homepage is strategically organized, featuring sections dedicated to highlighting completed projects, services offered, expert team members, and client testimonials. This thoughtful design aims to instill confidence in potential clients and partners. Functionality is a key focus of the Building Contractor WordPress theme. It includes features such as a project portfolio, allowing businesses to showcase their construction projects with details on scope, timeline, and success stories. The responsive design ensures optimal viewing on various devices, from desktops to tablets and mobile phones. Customization options empower businesses to personalize the theme according to their brand identity. From customizable color schemes and fonts to varied layout options, the theme allows for a distinctive and memorable online presence that aligns with the unique characteristics of each construction business. == Homepage Installation ==

Step-1. Go to Dashboard -- Pages -- Add new page -- Add templates (Home Page) -- Publish it.
Step-2. Go to Dashboard -- Settings -- Reading -- Click on static page -- Select home page -- Save changes.

Header Section
Step-1. Go to Dashboard -- Appearnace -- Customize -- Theme option panel -- Manage Header Section -- Fill the contact information. -- Publish it.

Slider Section
Step-1. Go to Dashboard -- Posts -- Add new Category -- Add new post -- Add title, content, featured image and select category -- Publish it.
Step-2. Go to Dashboard -- Appearnace -- Customize -- Theme option panel -- Manage Slider Section -- Select the category -- Publish it.

Our Services Section
Step-1. Go to Dashboard -- Post -- Add new category -- Publish it.
Step-2. Go to Dashboard -- Post -- Add new post -- Add content -- Add meta field Icon class -- Publish it.
Step-3. Go to Dashboard -- Appearnace -- Customize -- Theme option panel -- Manage Our Service Section
 -- Select the category -- Publish it.

Footer Section
Step-1. Go to Dashboard -- Appearnace -- Customize -- Theme option panel -- Manage Footer Section -- Add copyright text -- Publish it.
